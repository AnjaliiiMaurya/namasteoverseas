# Namaste Overseas 🧸✨

Welcome to the official repository for **Namaste Overseas**, a premium showcase website for high-quality plush toys manufactured in India.

## 🔗 Live Demo
Once GitHub Pages is enabled, the website will be accessible here:
👉 **[Live Website](https://anjaliilimaurya.github.io/namasteoverseas/)**

## 🚀 Features
- **Modern & Premium Design**: Clean layouts with curated color palettes and typography.
- **Single-page Application**: Extremely fast load times and clean navigation.
- **Responsive Layout**: Works seamlessly across mobile devices, tablets, and desktops.
- **Embedded Assets**: Self-contained design and media resources for instant unpacking and zero broken links.

## 🛠️ How to View Locally
1. Clone the repository:
   ```bash
   git clone https://github.com/AnjaliiiMaurya/namasteoverseas.git
   ```
2. Navigate into the directory and open `index.html` in any web browser.

## 📦 Project Structure
- `index.html`: The main bundled website landing page.
- `updated with image.html`: Backup of the original page bundle.
- `.gitignore`: Standard rules to ignore temporary system files.
